GitHub 文件加速（PHP版本）

本站仅供演示，如需可靠使用，请自行部署

输入示例：

分支源码：https://github.com/DUpdateSystem/UpgradeAll/archive/main.zip

release源码：https://github.com/DUpdateSystem/UpgradeAll/archive/0.1.5.zip

release文件: https://github.com/DUpdateSystem/UpgradeAll/releases/download/0.1.5/UpgradeAll_0.1.5.apk

分支文件：https://github.com/DUpdateSystem/UpgradeAll/blob/master/LICENSE

API请求：https://api.github.com/repos/reactphp/http/releases/latest

注意事项：

右键复制出来的链接都是符合标准的，但不支持项目文件夹

release、archive直接反代加速，文件会跳转至JsDelivr或cnpmjs.org加速下载

本项目为 gh-proxy 的PHP版本，由 JohnsonRan 基于 ReactPHP 开发
